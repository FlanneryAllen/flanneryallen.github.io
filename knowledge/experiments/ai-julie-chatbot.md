# AI Julie (Pre-Interview Chatbot)

Conversational AI experiment created for Meta application, trained on portfolio content to demonstrate fit for emerging experiences role.

## Platform

Built using Chatbase AI

## Purpose

Experimental approach to job applications: instead of static resume, created interactive chatbot that could answer questions about work, experience, and approach to emerging technology.

## Implementation

- Trained on complete portfolio content
- Embedded as live demo in application materials
- Interactive Q&A about work history, skills, and methodology
- Demonstrated AI implementation skills through the medium itself

## Outcome

Two experiments in using AI to demonstrate fit: showing rather than telling. Testing new approaches before they're required, as proof of adaptability and technical implementation capability.
